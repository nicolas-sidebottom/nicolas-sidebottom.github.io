Ranger Beads

This is an Arduino an C++ based project that uses an IMU to count steps and measure distance walked. I created this as a navigational tool to help support my passion and interest in the outdoors and cartography. While it is still a work in progress, the main components of the project have all been finalized with a custom PCB and 3-D printed case coming soon.

Satellite
Trajectory Mapper

This is a MATLAB based project that reads in a text file containing initial positions of satellites then uses Euler approximation to calculate thier trajectories. This program graphs the satelite trajectories around the globe, and in a seperate window plots satellite altitudes, speeds, and acceleration vs. time.

SOLIDWORKS
Turbojet Model

This is a SOLIDWORKS assembly of a simple can turbojet that I designed. It is made to function around an automotive turbocharger, and simple parts that can be purchased easily from automotive shops or Amazon. This design is meant to be cheap and easily constructed so that it can serve as an expedient test engine and learning tool.
